# Recommended Models for GenMedia

The following models are recommended for various media generation tasks within the GenMedia ecosystem.

## Imagen - Image Generation

*   **`imagen-3.0-fast-generate-001`**: The fastest Imagen 3 model, also known as Imagen Fast.
*   **`imagen-3.0-generate-002`**: The default Imagen 3 model.
*   **`imagen-4.0-generate-preview-05-20`**: The default Imagen 4 model.
*   **`imagen-4.0-ultra-generate-exp-05-20`**: The highest quality Imagen 4 experimental model.

## Veo - Video Generation

*   **`veo-2.0-generate-001`**: The default Veo 2 model.
*   **`veo-3.0-generate-preview`**: The newest Veo 3 model, which includes ambient audio and voice-overs. Use this for integrated video and audio; otherwise, combine Veo with Chirp 3 and Lyria.

## Lyria - Music Generation

*   **`lyria-002`**: The standard model for music generation.
*   **`lyria-3-clip-preview`**: Lyria 3 model for generating short music clips with lyric support.
*   **`lyria-3-pro-preview`**: Lyria 3 model for generating high-quality music with lyric support.
